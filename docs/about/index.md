# 关于

你好，我是苏易文。

这个网站用于长期整理和展示我的课程资料、学习笔记、实验报告、作业与项目，以及阶段性的学习总结。

## 这个网站想解决什么问题

分散在本地不同课程文件夹中的内容，往往不容易回顾，也不方便在需要的时候对外展示。这个网站希望把这些资料逐步整理成一个结构清楚、可持续更新、适合长期维护的学习档案站。

## 它会保持什么风格

- 内容真实，而不是过度包装
- 结构清楚，便于后续持续新增
- 页面简洁，阅读体验自然
- 更像个人学习资料站，而不是营销式主页

## 当前状态

目前是第一版框架阶段，已经建立首页、课程、项目 / 作业、笔记 / 报告和关于页的整体结构。后续会在不打乱主框架的前提下，逐步加入真实课程内容与精选材料。
