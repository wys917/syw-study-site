## 每门课程的统一结构

每门课后续都会尽量按照以下模块组织：

1. 课程简介
2. 资料导航
3. 重点内容
4. 项目 / 实验 / 作业
5. 学习总结

这种结构的好处是：
- 页面风格统一
- 后期加内容不需要反复改导航
- 能同时兼顾展示和检索
