# TOEFL Reading

> 这里整理托福阅读课的课程笔记、词根词缀、生词记录与阅读方法总结。为了避免所有 lecture 的内容堆在同一页，后续按 **Lecture** 分页整理，并在这里统一索引。

## 课程说明

这门课的笔记主要分成两类：

1. **Lecture Notes**
   - 按每一讲分别记录
   - 保留当次课的词根、分类、生词、替换词、逻辑词和老师补充

2. **专题整理**
   - 后续可把多个 lecture 里重复出现的内容，再抽成专题页
   - 例如：
     - 高频词根总表
     - 托福阅读同义替换词群
     - 因果 / 转折 / 比较逻辑词
     - 学科类背景词汇（人类学 / 生物 / 地理 / 历史）

## 使用方式

以后如果继续整理托福阅读课内容，建议直接按下面方式追加：

- 新的一讲：新建一个 `lecX.md`
- 同一讲补充：继续追加到对应 lecture 页面
- 多讲重复出现的重点：再额外整理到专题页

这样结构会更清楚，也更方便后面查找和复习。
